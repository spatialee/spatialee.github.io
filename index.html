When any subscriber makes a phone activity (e.g. call, message or data connection) the mobile network operator store the approximate location along with the time of the activity for billing purposes. The data is called Call Details Records (CDR). Data size is huge on country level population, and great insights can be extracted from it. Details here: Journal Article
Tech stack: Java for interpolation, Arcgis for landsat image classification, Postgis for OSM data processing
