Developement Work in Japan Space Systems

Background:

Japan Space System is responsible for managing and distributing ASTER satellite Datasets, The data set is huge with high spatial and temporal resolution and often difficult for non-experience users to search and retrieve their desired scene. In addition its also a leader in all remote sensing applications. The Internship was divided into two parts, System development and Remote Sensing.


(1) System Developement

 

Objective:

The objective was to define, design, test, and implement a new software application or program that allow non-experienced users to search for the scene of interest without any SQL experience.

Developing environment:

        OS: centos os

        Programing language: JAVA with PostgreSQL as a relational database management system

        Distribution method: Executable JAR
